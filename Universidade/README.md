# Projeto Universidade  
Análise de Dados com Power BI e Modelagem Dimensional

Este projeto faz parte do meu portfólio de estudos em Power BI. O objetivo é desenvolver um dashboard analítico completo para uma base fictícia de dados universitários, aplicando boas práticas de modelagem dimensional, transformação de dados, criação de medidas DAX e construção de insights visuais.


---

## Objetivo final do projeto  
O resultado esperado é um dashboard completo que permita analisar:  
- desempenho dos cursos  
- relação alunos x professores  
- volume de matrículas  
- desistências  
- indicadores acadêmicos  
- visão consolidada da universidade por área e período
